# 📊 Vishal Clothing Sales Analysis Dashboard — Excel

> Interactive sales intelligence across categories, states, channels, and customer segments using Excel.

---

## 📌 Overview

This Excel-based dashboard provides **end-to-end visibility into clothing sales performance** — covering category trends, gender distribution, regional demand, order status, and sales channels.

Built using **Pivot Tables, Charts, and Slicers**, this project demonstrates how Excel can be used as a powerful analytics tool without any external BI software.

**Key focus areas:** Category analysis · State-wise sales · Channel contribution · Gender insights · Order status · Monthly trends · Customer segmentation

## 🚀 Features & Highlights

### Business Problem

Retail businesses often lack a centralized view of **sales performance across categories, regions, and customer segments**. Raw transactional data makes it difficult to identify trends, top-performing areas, and customer behavior patterns.

## 💡 Key Business Insights

1. **Women dominate purchases (~99%)**
   → Marketing strategies should heavily focus on female customers.

2. **Maharashtra & Karnataka lead sales**
   → Strong regional demand; scope for expansion in underperforming states.

3. **Amazon & Myntra drive majority sales**
   → Marketplace dependency is high; diversification can reduce risk.

4. **92% orders are successfully delivered**
   → Strong operational efficiency with minimal returns/cancellations.

5. **Adults are the primary buyers**
   → Target audience is clearly defined; opportunity to expand into other age groups.

6. **Consistent monthly growth observed**
   → Sales trend shows positive momentum from Jan to March.
